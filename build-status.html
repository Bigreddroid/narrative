Remodel Vision (New Core Idea)

Dual Customer + Enterprise Bloomberg Terminal

Inspired by Bloomberg professional terminal (real-time news, data, analytics, tools, impact lenses on world events/markets) but built dual: accessible customer side + full enterprise depth. Dark terminal primary (rich black #0D1117, dense data viz) for pro/enterprise; light editorial paper variant (#F5F1EB paper + dark ink) for customer accessibility. Crimson #C80028 brand accent (logo wordmark, impact CTAs, "How this affects you", Media Bias pills) in both modes. Globe/map viz uses cyan glows (#00D4FF) on dark for hotspots/connections (or red in some mocks); same crimson on light. Key patterns from mocks: interactive world/globe with glowing dots, prominent "How this affects you" + bias, personalized feeds, dense cards, regional panels.

Web / Desktop: Same dual theme system (colors.js tokens portable). Dark for dense terminal views, light paper for editorial pages. Mobile-first but web/desktop terminal feel is stretch (globe + lists + impact panels matching the desktop mocks).

Simplify Significantly: Lean ingestion + light AI summaries/impact. No heavy 11-worker over-engineering.
